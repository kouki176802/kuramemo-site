# SNSアカウント接続準備

作成日: 2026-06-25

## 結論

投稿BOTの基礎はすでに完成している。現在は `Shadow Mode` として、SNS投稿案の生成、投稿キュー、承認、却下、予約変更、CSV出力、dry-run、Instagramカルーセル画像生成まで対応済み。

次に必要なのは、実アカウントへ合わせた運用設定。

## 現在できること

- 記事からSNS投稿案を生成
- X投稿文を生成
- Threads投稿文を生成
- Instagramカルーセル案を生成
- TikTok台本を生成
- 投稿キューへ登録
- CEO承認・却下
- 投稿予約変更
- 重複投稿防止
- 手動投稿用CSV出力
- InstagramカルーセルPNG生成
- dry-runで投稿処理の検証

## まだできないこと

実アカウントへの自動投稿は、各SNSのAPI認証と規約確認が必要。

- X: API利用とアクセストークンが必要。有料になる可能性が高い。
- Threads: Meta開発者設定、ユーザーID、アクセストークンが必要。
- Instagram: プロアカウント、Meta連携、画像の公開URL、アクセストークンが必要。
- TikTok: 初期は動画投稿なし。台本生成のみ。

## 無料で先にやる運用

初期はAPI自動投稿より、CSV出力から手動投稿する。

理由:

- API費用を抑えられる
- アカウントBAN・規約違反リスクを下げられる
- どの投稿型が伸びるか確認できる
- ASP審査用の実績を安全に作れる

## CEOが用意する情報

パスワードは不要。

- XアカウントURL
- ThreadsアカウントURL
- InstagramアカウントURL
- TikTokアカウントURL（使う場合）
- 各アカウントの表示名
- プロフィール文を差し替えるか
- 投稿してよい時間帯
- 1日の最大投稿数

## 初期推奨設定

### X

- 1日1〜3投稿
- ブログ記事への導線
- トレンド反応と短い選び方

### Threads

- 1日1〜2投稿
- Xより少し柔らかい文章
- 読者の悩み起点

### Instagram

- 週2〜3投稿
- カルーセル中心
- 保存されるチェックリスト型

### TikTok

- 当面は台本のみ
- 動画制作は後回し

## 投稿BOTの実装段階

### Phase 1: 手動投稿BOT

状態: 今すぐ可能。

内容:

- BOTが投稿案を作る
- CSVで出力
- CEOまたは手動作業で投稿
- 反応を記録

### Phase 2: 半自動BOT

条件:

- SNSアカウントURL確定
- 投稿ルール確定
- 10〜30投稿の反応確認

内容:

- 投稿キューを毎日生成
- CEOが承認
- 手動または一部APIで投稿

### Phase 3: API自動投稿BOT

条件:

- API費用を許可
- 各SNSの開発者設定完了
- アクセストークン取得
- 投稿ルールと停止条件が確定

内容:

- 承認済み投稿を予約時刻に自動投稿
- 失敗時に再試行
- 投稿履歴を保存
- 指標を取り込む

### Phase 4: 収益最適化BOT

条件:

- ASPリンク取得
- ブログ公開
- SNSからブログ流入が発生
- クリック・成果データが取れる

内容:

- 反応が良いテーマを増やす
- 成果が低い投稿型を止める
- 記事タイトルやCTAを改善する
- 案件ごとの収益性を評価する

## 次にやること

1. 各SNSアカウントURLを記録
2. プロフィール文を入れる
3. 固定投稿を作る
4. BOTで投稿案をCSV出力
5. まず手動投稿で10〜30件テスト
6. 伸びた型だけ自動化候補にする

## 注意

最初から完全自動投稿にしない。特に広告リンク付き投稿、美容・健康系投稿、比較ランキングはCEO確認を通す。
