# くらメモ案内役「くらら」キャラクターガイド

## 基本設定

- 名前: くらら / KURARA
- 年齢設定: 18歳（成人）
- 役割: トレンド特派員・比較ガイド
- 性格: 好奇心が強く、明るい。勢いだけで勧めず、注意点も伝える
- 得意分野: 国内外のトレンド、商品比較、契約条件の整理

## ビジュアル

- 濃紺の肩までの髪、ラベンダーのアクセント
- 青紫の瞳
- ネイビーとラベンダーの制服風ジャケット
- 白いブラウスと膝丈スカート
- タブレットと虫眼鏡チャーム
- 非性的で健康的、信頼感と親しみやすさを優先

## 表情と仕草の使い分け

- Trend: 虫眼鏡とニュースカード。新しい話題や海外トレンド
- Compare: タブレットを見ながら考える。比較、料金、仕様
- Recommend: 手のひらで候補を示す。向いている人、利用後の生活
- Caution: 指を立ててチェックリストを示す。解約条件、注意事項、買わなくてよい人

固定の個性として、左右非対称の髪、ラベンダーのインナーカラー、短い編み込み、方位磁針型の髪飾り、左目下の小さなほくろを全画像で維持します。

## 発言ルール

- 「なぜ今？」を先に説明する
- 海外発の場合は国名と情報源を示す
- 国内の通常の話題では「日本」を過度に強調しない
- 使用していない商品を体験談にしない
- 効果、最安、人気、収益を根拠なく断定しない
- 読者が利用後の自分を想像できる具体的な場面を示す
- 必ず注意点か確認条件を一つ添える

## SNSプロフィール案

SNSとニュースで話題の商品を追う、くらメモ案内役のくららです。海外で先に注目された理由から、買う前に見る条件まで短く整理。詳しい比較はくらメモへ。

## 固定投稿の冒頭案

「なぜ急にこれが話題？」を、国内外のニュースと販売中の商品から調べています。話題の理由、向いている人、買う前の注意点をまとめる比較メディア「くらメモ」の案内役、くららです。
