# ASP登録準備ガイド

作成日: 2026-06-25

## 結論

今すぐASP登録できなくても問題ない。先に媒体プロフィール、SNSプロフィール、固定ページ、記事サンプルを整える方が審査上も運用上も安全。

登録は、空の媒体で急ぐよりも、最低限の公開実態を作ってから行う。

## 登録前に用意するもの

### 必須

- 媒体名
- 媒体説明文
- 公開SNSまたはサイトURL
- 運営者情報
- 広告表記方針
- プライバシーポリシー
- 問い合わせ先

### あると強い

- 5〜10本のサンプル記事
- SNS投稿10〜30件
- 固定投稿
- 編集方針ページ
- 広告掲載方針ページ
- 扱わないジャンルの明記

## ASP登録時の媒体説明文

選び方メモは、ニュース、季節トレンド、新商品、話題のサービスをもとに、読者の購買判断を支援するメディアです。対象ジャンルはAI・ガジェット、美容・フィットネス、季節・暮らしを中心とし、公式情報・販売条件・価格・在庫・保証・解約条件などを整理して紹介します。

記事では、商品を無理に勧めるのではなく、買うべき人、買わなくてよい人、無料または手持ち品で代用できる方法、比較時に見るべきポイントを明確にします。アフィリエイト広告を利用する場合は、広告であることを読者が認識しやすい位置に表示します。

AIは情報整理、下書き、比較表作成、SNS展開、改善分析に利用しますが、公開前に広告表示、出典、禁止表現、商品条件を確認する運用を行います。医療・投資助言・災害被害・事件事故・誇大な健康効果・収益保証など、読者の不安を過度に利用する内容は扱いません。

## 登録順序

### 1. 楽天アフィリエイト

状態: 登録済み。媒体情報は後で確認。

向いている用途:

- 生活用品
- 季節用品
- 美容用品
- フィットネス用品
- ガジェット周辺機器

注意:

- SNS掲載条件を媒体ごとに確認する
- DM、非公開アカウント、メール等への無差別共有は避ける
- 商品価格や在庫はリンク先確認に寄せる

### 2. もしもアフィリエイト

状態: 未登録。

向いている用途:

- 物販
- Amazon・楽天・Yahoo系の導線整理
- 初期の物販比較

注意:

- 登録媒体以外での広告掲載をしない
- Amazon提携は媒体実態ができてから申請する

### 3. A8.net

状態: 未登録。

向いている用途:

- 美容
- 健康食品
- フィットネス
- 通信、ガジェット系サービス
- 転職やスクール系を将来扱う場合

注意:

- 健康効果、痩身、収益性などの表現ルールを厳しく見る
- SNS掲載可否は案件ごとに確認する

### 4. バリューコマース

状態: 未登録。

向いている用途:

- 大手EC
- 通販
- ガジェット
- 旅行やサービス系への将来拡張

注意:

- 媒体審査用に公開ページを整えてから登録する

### 5. Amazonアソシエイト

状態: 未申請推奨。

向いている用途:

- ガジェット
- 生活用品
- 書籍
- 日用品

注意:

- 空のサイトや薄いSNSで急がない
- 公開記事・SNS投稿・固定ページを作ってから申請する
- 有料広告やSNS投稿での扱いは最新規約を都度確認する

## 登録時に入力する推奨カテゴリ

ASP側の選択肢に近いものを選ぶ。

- ショッピング
- ライフスタイル
- IT・ガジェット
- 美容
- 健康・フィットネス
- 生活用品

## 申請前チェック

- [ ] 媒体名が決まっている
- [ ] 公開URLがある
- [ ] プロフィール文が入っている
- [ ] 広告表記がある
- [ ] プライバシーポリシーがある
- [ ] 問い合わせ先がある
- [ ] 投稿または記事が5本以上ある
- [ ] NGジャンルを扱っていない
- [ ] AI利用方針を説明できる

## 案件を有効化する条件

以下を満たすまで、`data/offers.csv` の `status` は `active` にしない。

1. ASPまたは広告主の提携承認がある
2. 掲載媒体が登録済み
3. 成果条件と否認条件を確認済み
4. SNS掲載可否を確認済み
5. 広告表記方法を確認済み
6. 実際のアフィリエイトURLを取得済み
7. リンク先の商品・価格・地域条件を確認済み
8. 最終確認日を記録済み

## 登録できない日の作業

登録できない日は、次の順で進める。

1. 媒体プロフィールを作る
2. SNSプロフィールを作る
3. 固定ページを整える
4. 記事テンプレートを作る
5. SNS投稿テンプレートを作る
6. サンプル記事を作る
7. サンプル投稿を作る
8. 審査用チェックリストを埋める
