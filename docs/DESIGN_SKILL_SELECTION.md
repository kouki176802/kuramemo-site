# デザインスキル選定メモ

## 選定結果

このプロジェクトでは `editorial` デザインスキルを採用する。

参照先:

`/Users/matsumotokouki/.codex/skills/editorial/SKILL.md`

## 比較した候補

| スキル | 向いている点 | 懸念 | 判定 |
|---|---|---|---|
| editorial | 雑誌風、読み物、比較記事、信頼感を作りやすい | セリフ体を強く使いすぎると日本語で読みにくい | 採用 |
| publication | 書籍・雑誌風で強い個性が出る | 色と表現が少し派手で、商品比較サイトには主張が強い | 不採用 |
| premium | Apple風で洗練される | 商品比較・ランキング・記事導線にはやや無機質 | 不採用 |
| refined | 上品でミニマル | 商品カードや比較表の情報密度を出しにくい | 保留 |
| bento | カード整理、比較ブロックに強い | 記事メディアとしての読ませる力が弱い | 部分採用候補 |
| professional | ビジネス感・信頼感が出る | 家電量販店っぽくなりやすく、売り込み感が出る | 不採用 |
| clean | 読みやすい | 既存サイトとの差分が弱く、印象に残りにくい | 不採用 |

## 今後のデザイン方針

`editorial` をベースに、次の方向でサイトを作り直す。

- 売り込み感を抑えた比較メディア
- 雑誌風の見出しと余白
- 商品カードは読み物の中に自然に置く
- 比較表、ランキング、注意点を見やすくする
- CTAは強すぎず、判断材料の延長として置く
- 日本語本文は可読性優先で、セリフ体の使いすぎは避ける

## 実装時の注意

- 見出しや特集枠は editorial の雰囲気を使う
- 本文は日本語可読性のため sans-serif を基本にする
- 商品カードは bento 的な整理も部分的に使う
- アクセシビリティは WCAG 2.2 AA を目標にする
- モバイルでは装飾より読みやすさを優先する
