# LINE・メール導線

登録直後に商品を売らず、用途整理から比較ページへ戻す。配信原稿の台帳は`data/lifecycle_messages.csv`。

## 導線

SNS → 入口記事 → 比較ページ → LINEまたはメール → 条件更新・季節通知 → 比較ページ → 広告リンク

## 自動化条件

- 本人が明示的に登録した読者だけに送る
- 配信停止リンクを常設する
- LINEとメールへ同じ内容を同時連投しない
- クリック済みでも成約有無を断定しない
- 価格、在庫、特典は配信本文へ固定せず販売ページ確認へ誘導する
- 1テーマ週2通を上限とする

## BOT担当

- CRM管理BOT: 同意、配信停止、セグメントを管理
- 編集BOT: 記事から配信文へ変換
- 配信BOT: 時刻、上限、重複を検査
- 分析BOT: 開封、クリック、解除を集計
- 品質BOT: 誇張、煽り、誤認表現を止める

配信サービス契約前は、原稿生成とCSV出力までを無料で行う。実送信はサービス契約と同意管理が整ってから有効化する。
