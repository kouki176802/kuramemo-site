# ランキング形式の運用ルール

作成日: 2026-06-25

## 結論

ランキング形式は使ってよい。

ただし「総合1位」「絶対おすすめ」「売れている順」だけで押すと売り込み感が強くなり、比較メディアの信頼を落とす。初期は、用途別・条件別ランキングを中心にする。

## 使ってよいランキング

### 用途別ランキング

例:

- 初心者向けAIツール3選
- 持ち運び重視のモバイルバッテリー3選
- 一人暮らし向け暑さ対策グッズ3選

### 条件別ランキング

例:

- 軽さ重視
- 価格重視
- 続けやすさ重視
- 省スペース重視
- 解約しやすさ重視

### チェックリスト型ランキング

例:

- 買う前に見るべきポイントTOP5
- 失敗しやすいポイントTOP5
- 初心者が確認したい順番TOP5

### SNS向けランキング

例:

- まず見るべき3つ
- 買う前チェック5選
- 初心者が失敗しやすい3パターン

## 避けるランキング

- 報酬が高い順
- 根拠なしの売上順
- 「全員におすすめ」ランキング
- 偽の口コミランキング
- 使っていない商品の体験ランキング
- 医療効果、痩身効果、収益性を断定するランキング

## ランキングの基準

ランキング記事には、必ず評価基準を書く。

標準評価基準:

1. 用途への合いやすさ
2. 価格と総費用
3. 仕様・互換性
4. 保証・返品・解約条件
5. 続けやすさ・使いやすさ
6. 入手性
7. アフィリエイト報酬

報酬は最後。報酬だけで順位を上げない。

## タイトル例

### 良い

- 初心者向けAIツール3選｜有料化前に見るポイント
- 急速充電器おすすめタイプ別3選｜スマホ・PC別に比較
- 家トレ用品は何から？初心者が選びやすい順番TOP5
- 暑さ対策グッズ用途別ランキング｜室内・外出・就寝用

### 悪い

- 絶対買うべきAIツールランキング
- これを買わないと損
- 全員におすすめの最強商品
- 効果抜群ランキング
- 必ず稼げる副業ツールTOP10

## 記事テンプレート

```text
タイトル:
{テーマ}ランキング｜{用途/条件}別に選び方を整理

広告表記:
この記事には広告リンクを含む場合があります。

導入:
{テーマ}は種類が多く、ランキングだけで選ぶと失敗することがあります。
この記事では、{評価基準}をもとに、用途別に選びやすい候補を整理します。

評価基準:
1. {基準1}
2. {基準2}
3. {基準3}
4. 価格・保証・返品条件

ランキング:
1. {用途A向け}
2. {用途B向け}
3. {用途C向け}

買わなくていい人:
- {条件}
- {条件}

注意点:
- 価格、在庫、条件は変わる
- リンク先で最新情報を確認
- 未使用商品の体験談ではない場合は明記

まとめ:
ランキングは目安です。自分の用途、予算、条件に合うかを確認して選んでください。
```

## SNSテンプレート

### X

```text
{テーマ}、ランキングで見るなら「総合1位」より用途別が分かりやすいです。

1. {用途A向け}
2. {用途B向け}
3. {用途C向け}

全員に同じ正解はないので、価格・保証・使う場面まで見るのが安全。
※広告を含む場合があります
```

### Instagram

```text
1枚目: {テーマ} 用途別ランキング
2枚目: ランキング基準
3枚目: {用途A向け}
4枚目: {用途B向け}
5枚目: {用途C向け}
6枚目: 買わなくていい人
7枚目: 詳細は比較ページへ
```

## BOTへの指示

ランキングを生成する場合、BOTは次を必須にする。

- 評価基準を明記
- 買わなくていい人を入れる
- 広告表記を入れる
- 報酬順にしない
- 実リンク未承認なら商品名を断定しない
- 未使用商品は体験談化しない
- 健康・金融・災害・事件事故では自動ランキング化しない
