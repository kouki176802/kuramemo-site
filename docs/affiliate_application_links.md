## 申請後に必要なもの

1. ASP名
2. プログラム名またはプログラムID
3. 審査状態（申請中・承認・否認）
4. 承認後に発行された広告リンクまたはバナーコード
5. 広告掲載を許可された媒体（Web・X・Instagram等）

この5項目が揃った案件だけを「提携済み広告」としてサイトへ反映します。
