# 媒体プロフィール一式

作成日: 2026-06-25

## 媒体名

選び方メモ（仮）

## 一言コンセプト

話題の商品やサービスを、比較表・選び方・向き不向きで整理する比較メディア。

## 事業タイプ

トレンド連動型の比較メディア。

比較ページを中心にしながら、SNS投稿、トレンド記事、個別チェック記事を組み合わせて運営する。

## 読者への約束

選び方メモは、ニュースを煽りとして使わず、読者が冷静に商品やサービスを選べるように整理します。

- まず「買う必要がある人・ない人」を分ける
- 無料で代用できる方法があれば先に示す
- 商品を紹介する場合は選び方を先に説明する
- 広告リンクがある場合は分かりやすく表示する
- 未使用商品を「使った」とは書かない

## 媒体説明文: 短文

季節、暮らし、美容、フィットネス、AI、ガジェットなどの話題を、比較表と選び方で整理する比較メディアです。

## 媒体説明文: ASP登録用

選び方メモは、ニュース、季節トレンド、新商品、話題のサービスをもとに、読者が商品・サービスを比較しやすい形へ整理するメディアです。対象ジャンルはAI・ガジェット、美容・フィットネス、季節・暮らしを中心とし、公式情報・販売条件・価格・在庫・保証・解約条件などを比較表とチェックリストで紹介します。

記事では、商品を無理に勧めるのではなく、買うべき人、買わなくてよい人、無料または手持ち品で代用できる方法、比較時に見るべきポイントを明確にします。アフィリエイト広告を利用する場合は、広告であることを読者が認識しやすい位置に表示します。

AIは情報整理、下書き、比較表作成、SNS展開、改善分析に利用しますが、公開前に広告表示、出典、禁止表現、商品条件を確認する運用を行います。医療・投資助言・災害被害・事件事故・誇大な健康効果・収益保証など、読者の不安を過度に利用する内容は扱いません。

## SNSプロフィール文

### X

話題の商品・サービスを比較表と選び方で整理。AI/ガジェット、美容/フィットネス、暮らし用品の「買う前チェック」を発信。広告を含む場合があります。

### Threads

ニュースや流行を、比較して選べる形に整理するメディアです。買うべき人・買わなくていい人・比較ポイントを短く整理します。広告を含む場合があります。

### Instagram

話題の商品・季節アイテム・AI/ガジェットを、比較カードで整理。買う前に見るチェックポイントを発信します。広告を含む場合があります。

### TikTok

話題の商品やサービスを「買う前に見るべきポイント」で短く解説。煽りではなく、選び方重視。広告を含む場合があります。

## SNS固定投稿案

選び方メモは、話題のニュースや流行を「何を買うべきか」ではなく、まず「本当に買う必要があるか」から整理する買い物判断メディアです。

主に扱うテーマ:

- AI・ガジェット
- 美容・フィットネス
- 季節・暮らし用品

方針:

- 広告は広告と分かるように表示
- 未使用品を使ったとは書かない
- 無理に買わせる記事は作らない
- 買わない選択肢も出す

## 想定読者

### 主要読者

- 話題の商品が気になるが、買う前に判断材料がほしい人
- SNSで見た商品をその場で買う前に、比較ポイントを確認したい人
- 忙しくて公式情報や条件を読み込む時間がない人
- 新商品・セール・季節用品に反応しやすいが、失敗買いを減らしたい人

### 補助読者

- AIツールやガジェットに興味がある初心者
- メンズ美容・筋トレ・健康習慣を始めたい人
- 一人暮らしや季節の生活用品を効率よく選びたい人

## 扱うテーマ

### 初期公開対象

- AIサービス、生成AIツール、SaaS
- スマホ周辺機器、充電器、イヤホン、PC周辺機器
- メンズ美容、スキンケア、ヘアケア
- 筋トレ用品、プロテイン、ウェア
- 暑さ、寒さ、梅雨、新生活、家事効率、収納、調理用品

### 監視のみ

- 転職
- 食品
- 旅行
- エンタメ
- 海外サービス

### 自動収益化しないテーマ

- 医療、診断、治療、処方
- 投資助言、金融商品の推奨、暗号資産
- 災害、事故、死亡、犯罪
- 芸能人の炎上・スキャンダル
- 未成年者の不安を利用する商品
- 根拠のない健康効果、収益保証、誇大なビフォーアフター

## ブランドトーン

- 先に結論
- 短く、分かりやすく
- 焦らせない
- 断定しすぎない
- 事実、推測、編集判断を分ける
- 「買え」ではなく「選び方」を出す

## プロフィール登録時の仮情報

登録時に未確定なら、以下を使う。

- 媒体名: 選び方メモ
- 運営者名: 選び方メモ編集部
- サイトURL: 未定
- X URL: 未定
- Threads URL: 未定
- Instagram URL: 未定
- TikTok URL: 未定
- お問い合わせURL: 未定
- 主カテゴリ: ショッピング / ライフスタイル / IT・ガジェット
- 副カテゴリ: 美容 / 健康・フィットネス / 生活用品

## 後でCEOが決める項目

- 正式な媒体名
- 公開用の運営者名
- サイトURL
- SNSアカウントURL
- 問い合わせフォームURL
- 顔出し・個人名を出すかどうか
