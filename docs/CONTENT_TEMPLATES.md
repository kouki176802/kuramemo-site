# 記事・SNSテンプレート

作成日: 2026-06-25

## 共通方針

記事は、読者が自然に商品を手に取りやすい導線で書く。ただし、恐怖、誤認、偽の緊急性、架空の口コミ、未使用商品の体験談は使わない。

心理設計は、煽りではなく「判断負荷を下げる」ために使う。

## 行動心理の使い方

### 使用後の自分を想像できる文章

商品・サービスの機能を説明した直後に、読者が利用後の生活を具体的に思い浮かべられる短い情景を入れる。

- 「朝・外出前・仕事中・帰宅後」など場面を一つ決める
- 使用前の小さな手間が、使用後にどう変わるかを書く
- 周囲からの見え方は「そう見られることが期待できる」「本人がそう感じやすくなる」のように可能性で表現する
- 気持ちの変化は「不安が消える」と断定せず、「気にする時間が減る」「服装や予定に意識を向けやすい」と具体化する
- 未使用商品について、使用者本人の体験談のように書かない
- 恋愛成就、好意、収入、健康効果など、商品だけでは保証できない結果を約束しない

基本構成:

1. 使用する場面
2. 目に見える変化または減る手間
3. 周囲に与えうる印象
4. 本人が感じうる気持ち
5. 個人差・条件の注意

例（脱毛）:

> 朝の着替えや外出前に、処理できているかを何度も確認する時間が減るかもしれません。肌を見せる服装でも身だしなみを整えている印象につながり、自分自身も視線より予定や会話に意識を向けやすくなります。仕上がりや必要回数には個人差があります。

### 使ってよい

- 損失回避: 失敗買いを避けるチェックポイントを出す
- 社会的証明: 公式ランキング、レビュー数など確認可能な情報に限定する
- 希少性: 公式に確認できる在庫・期間だけを書く
- アンカリング: 価格帯や機能差を比較表で示す
- 選択肢の整理: 初心者向け、コスパ重視、こだわり派に分ける
- フット・イン・ザ・ドア: まず無料確認、次に低価格商品、高価格商品は最後

### 使ってはいけない

- 嘘の期限
- 嘘の在庫
- 架空の口コミ
- 「全員買うべき」
- 「買わないと損」とだけ煽る
- 不安や病気を過度に刺激する
- 未使用商品の体験談化

## 記事テンプレート: 集客記事

目的: SNSや検索から入口を作る。売り込みは弱め。

```text
タイトル:
【話題】{トレンド名}で何が変わる？買う前に見るポイント

導入:
{トレンド名}が話題になっています。
ただ、話題だからといって全員がすぐ買う必要はありません。
この記事では、影響を受ける人、買う前に確認したいポイント、代用できる方法を整理します。

広告表記:
この記事には広告リンクを含む場合があります。

1. 何が話題になっているか
- 公式情報
- 変更点
- 対象者

2. 影響を受ける人
- 該当する人
- 該当しにくい人

3. 今すぐ買う必要があるか
- すぐ必要
- 急がなくてよい
- まず無料で確認

4. 買うなら見るべきポイント
- 価格
- 互換性
- 保証
- 解約・返品
- 在庫

5. よくある失敗
- 仕様違い
- サイズ違い
- 維持費
- 解約条件

6. まとめ
- 買うべき人
- 買わなくていい人
- 次に見る記事
```

## 記事テンプレート: 比較記事

目的: 商品選びの判断を助ける。アフィリエイトクリックにつなげる。

```text
タイトル:
{商品カテゴリ}の選び方｜初心者・コスパ・こだわり派で比較

導入:
{商品カテゴリ}は種類が多く、価格だけで選ぶと失敗しやすいジャンルです。
この記事では、用途別に見るべきポイントを整理します。

広告表記:
この記事には広告リンクを含みます。

1. 先に結論
- 初心者向け
- コスパ重視
- こだわり派

2. 比較表
- 価格
- 主な機能
- 向いている人
- 注意点
- 公式/販売ページ

3. 選び方
- 用途
- 予算
- 使用頻度
- 保証
- ランニングコスト

4. 買わなくていいケース
- 既に代用品がある
- 使用頻度が低い
- 維持費が合わない

5. よくある質問

6. まとめ
```

## 記事テンプレート: 比較サイト主力ページ

目的: SNS・検索・トレンド記事から受ける中核ページ。

```text
タイトル:
{商品カテゴリ}比較｜用途別に選び方と注意点を整理

導入:
{商品カテゴリ}は種類が多く、話題の商品だけで選ぶと失敗しやすいジャンルです。
この記事では、用途別に向いている選び方、比較表、買わなくていいケースを整理します。

広告表記:
この記事には広告リンクを含みます。

1. 先に結論
- 初心者向け
- コスパ重視
- 機能重視
- 買わなくていい人

2. 比較表
| 商品/サービス | 価格帯 | 向いている人 | 注意点 | 確認リンク |

3. 用途別の選び方
- 初めて使う人
- 安く済ませたい人
- 長く使いたい人
- 失敗したくない人

4. 比較するときのチェックポイント
- 価格
- 仕様
- 互換性
- 保証
- 返品/解約
- 維持費

5. 買わなくていいケース
- 手持ち品で足りる
- 使用頻度が低い
- 条件が合わない

6. よくある質問

7. 更新履歴

8. まとめ
```

## 記事テンプレート: 収益記事

目的: 成約に近い読者を受け止める。誤認を避けながら背中を押す。

```text
タイトル:
{商品名/サービス名}は誰向け？申し込み前に確認するポイント

導入:
{商品名}は{特徴}で注目されています。
ただし、すべての人に向くわけではありません。
申し込み前に、向いている人、注意点、代替候補を確認します。

広告表記:
この記事には広告リンクを含みます。

1. 先に結論
- 向いている人
- 向いていない人
- 確認すべき条件

2. 特徴
- 公式情報に基づく特徴
- できること
- できないこと

3. 注意点
- 価格
- 解約
- 対応環境
- 返品/保証
- 成果条件

4. 他候補との違い

5. 申し込み前チェックリスト

6. リンク先で確認すること

7. まとめ
```

## 記事テンプレート: ランキング記事

目的: 比較ページの補助として、用途別・条件別に選びやすくする。報酬順ランキングにしない。

```text
タイトル:
{テーマ}ランキング｜{用途/条件}別に選び方を整理

導入:
{テーマ}は種類が多く、ランキングだけで選ぶと失敗することがあります。
この記事では、{評価基準}をもとに、用途別に候補を整理します。

広告表記:
この記事には広告リンクを含む場合があります。

1. 評価基準
- 用途
- 価格
- 仕様
- 保証/返品/解約
- 使いやすさ

2. 用途別ランキング
- {用途A向け}
- {用途B向け}
- {用途C向け}

3. 買わなくていい人

4. 注意点

5. 比較ページへのリンク

6. まとめ
```

## SNSテンプレート: Threads

```text

見るところは価格より先に、
・自分の用途に合うか
・今あるもので代用できないか

ここを見てから選ぶと失敗しにくいです。
詳しくは記事にまとめます。
※広告を含む場合があります
```

## CTAテンプレート

### 弱いCTA

気になる人は、リンク先で最新価格と条件を確認してください。

### 中くらいのCTA

購入前に、価格・在庫・保証・返品条件だけはリンク先で確認しておくと安心です。

### 強めだが許容されるCTA

条件が合う人は、キャンペーンや在庫が変わる前に公式ページで最新情報を確認してください。

### 使わないCTA

- 今すぐ買わないと損です
- 全員買うべきです
- 絶対に失敗しません
- 効果は保証できます
- 残りわずかです（公式確認がない場合）
