# 公開前チェックリスト

作成日: 2026-06-25

## サイト公開前

- [x] 正式な媒体名を決める（くらメモ）
- [ ] ドメインを決める
- [x] ローカルWordPressを用意する
- [x] トップページを作る
- [x] カテゴリページを作る
- [x] 比較ページをローカルWordPressへ同期する
- [x] このサイトについてをローカル公開する
- [x] 広告掲載方針をローカル公開する
- [x] 編集方針をローカル公開する
- [x] プライバシーポリシーをローカル公開する（事業者情報は外部公開前に最終化）
- [x] 免責事項をローカル公開する
- [ ] 問い合わせフォームを作る

## SNS公開前

- [ ] Xプロフィール文を設定
- [ ] Threadsプロフィール文を設定
- [ ] Instagramプロフィール文を設定
- [ ] 固定投稿を作る
- [ ] 広告を含む場合の表記を確認
- [ ] 手動投稿テストを10本行う
- [ ] 投稿URLを記録する

## ASP登録前

- [ ] サイトURLがある
- [ ] SNS URLがある
- [ ] 固定ページが公開されている
- [ ] 比較ページが5本以上ある
- [ ] 広告方針がある
- [ ] AI利用方針がある
- [ ] 禁止ジャンルを扱っていない
- [ ] サンプル投稿がある

## 案件掲載前

- [ ] ASP承認済み
- [ ] 媒体登録済み
- [ ] 実アフィリエイトURL取得済み
- [ ] SNS掲載可否確認済み
- [ ] 成果条件確認済み
- [ ] 否認条件確認済み
- [ ] 禁止表現確認済み
- [ ] `data/offers.csv` を `active` に更新

## API自動投稿前

- [ ] 手動投稿で10〜30件テスト済み
- [ ] 投稿型ごとの反応を確認済み
- [ ] API費用上限を決めた
- [ ] 各SNSの認証情報を取得済み
- [ ] テスト投稿で削除手順を確認済み
- [ ] 停止条件を確認済み

## 自動化してはいけない状態

- サイトURLが未定
- 広告表記が未確定
- ASP承認がない
- 投稿先SNSの規約を確認していない
- 健康・金融・災害・事件事故テーマが混ざっている
- 実リンクと仮リンクが混在している
- CEOが停止判断している
