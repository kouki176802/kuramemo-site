# SNS信頼導線設計

作成日: 2026-06-25

## 目的

SNS投稿を単なる宣伝にせず、比較ページへ進む前の信頼形成として使う。

## 投稿ステージ

### 1. 認知投稿

目的:

- 話題に気づいてもらう
- 自分ごと化してもらう

例:

- AIツール、話題だけど有料化前に見るところがあります
- 急速充電器はワット数だけで選ぶと失敗しやすいです

### 2. 信頼投稿

目的:

- 売り込みではなく、判断材料を出す
- 買わない選択肢を示す

例:

- モバイルバッテリーは大容量ほど正解ではないです
- メンズスキンケアは最初からフルセット不要です

### 3. 比較投稿

目的:

- 用途別・条件別に整理する
- 保存される投稿を作る

例:

- 急速充電器は小型/多ポート/高出力で見る
- 暑さ対策グッズは室内/外出/就寝で分ける

### 4. 行動投稿

目的:

- 比較ページへ誘導する
- 購入ではなく確認行動へ促す

例:

- 詳細な比較表は記事にまとめます
- 買う前に、対応機種と保証だけ確認してください

## DRM 6ステップ対応

| DRM要素 | SNS投稿での役割 | 例 |
|---|---|---|
| 目的 | こうなれる/避けられるを見せる | 失敗買いを減らす |
| 信用 | 買わない選択肢を出す | 手持ち品で足りる人もいる |
| 問題点 | 悩みを代弁する | ワット数だけで選ぶと失敗しやすい |
| 手段 | 比較表・ランキングへつなぐ | 用途別に見る |
| 投資 | 価格・維持費・解約を見る | 月額固定費に注意 |
| 行動 | 最新条件確認へ促す | リンク先で確認 |

## 初期30日の運用

### 1〜7日目

- リンクなし投稿中心
- 比較視点を出す
- 返信で信頼を作る

### 8〜14日目

- 比較投稿を増やす
- 用途別ランキングを入れる
- プロフィールへの誘導を弱く入れる

### 15〜30日目

- 比較ページがある場合のみリンク投稿を開始
- 広告リンクではなく、比較ページへの誘導を中心にする
- 反応が良いテーマを比較ページに反映

## 投稿ごとの目的設定

BOTは投稿ごとに、次のどれが目的かを記録する。

- 認知
- 信用
- 問題点
- 手段
- 投資
- 行動

目的が曖昧な投稿は出さない。

## 初期投稿比率

| 投稿タイプ | 比率 |
|---|---:|
| 認知投稿 | 30% |
| 信頼投稿 | 30% |
| 比較投稿 | 30% |
| 行動投稿 | 10% |

最初から行動投稿を増やしすぎない。

## プラットフォーム別

### X

- 認知投稿と比較投稿を多め
- トレンドに即反応
- リンクは貼りすぎない

### Threads

- 信頼投稿を多め
- 少し会話調
- 体験談風にしすぎない

### Instagram

- 比較投稿中心
- 保存されるカード
- 用途別ランキングとチェックリスト

### TikTok

- 認知投稿と比較投稿の台本
- 動画投稿は後回し

## 返信運用

返信は、販売ではなく信頼作り。

やる:

- 短く補足する
- 相手の文脈に合わせる
- 比較視点を1つだけ置く
- リンクを貼らない

やらない:

- 商品リンクを貼る
- いきなり勧誘する
- 相手を否定する
- 煽る
- コピペする

## 投稿評価

インプレッションだけで判断しない。

重視:

1. 保存
2. プロフィールクリック
3. リンククリック
4. 比較ページ滞在
5. アフィリエイトクリック
6. 成果

インプレッションが多くても、比較ページに来ない投稿は増やしすぎない。
