# 海外展開

日本サイトの翻訳ではなく、国ごとに商品、価格、配送、広告規制を持つ別メディアとして展開する。市場台帳は`data/international_market_backlog.csv`。

## 日本のアフィリエイトを海外向けに使えるか

一部のデジタルサービスは可能だが、日本の物販リンクをそのまま海外読者へ出す方法は基本的に弱い。海外読者が購入できない、送料が高い、通貨と在庫が違うためである。

物販はAmazonなど対象国のプログラムへ国別登録し、現地商品へ差し替える。SaaSはImpact、PartnerStackなどグローバル案件を使い、対象国とトラッキング条件を案件ごとに確認する。

## 開始条件

- 日本版で月100クリック以上の勝ちテーマが3つある
- 英語版は別ドメインまたは`/en/`でURLを分離
- 現地価格、在庫、配送、返品を取得できる
- 国別の広告開示文を用意できる
- 翻訳ではなく現地検索意図でタイトルと商品を選び直す

## 最初の市場

英語圏のAI SaaSから始める。物流がなく、同じ比較ロジックを再利用しやすいためである。次に米国ガジェット、旅行eSIMへ広げる。

海外版BOTは日本版とDBを分け、通貨、タイムゾーン、ASP、法定表記を混ぜない。
