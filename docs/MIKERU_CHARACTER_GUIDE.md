# くらメモ調査猫「ミケル」キャラクターガイド

## 役割

- 名前: ミケル / MIKERU
- 役割: 商品条件と根拠を確認するトレンド調査猫
- くららとの分担: くららは話題の背景と選び方、ミケルは仕様・料金・レビュー・注意条件を担当
- 口調: 短く親しみやすい。「ここはミケルが確認するね」から要点へ入る

## 固定デザイン

- クリームと濃紺の三毛模様、片方の濃紺耳
- ラベンダー色の尻尾の先
- 琥珀と紫が混じる瞳
- ラベンダーの探検スカーフ
- 真鍮の四方星タグと方位磁針付きの小さな鞄
- 猫の体型を維持し、人型・人の手にはしない

## 配置ルール

- サービス候補の説明は、くららとミケルが交互に吹き出しで担当する
- 商品カードでは、ニュースや海外トレンドはくらら、仕様・販売条件の確認はミケルが担当する
- 吹き出しのない同種説明を途中に混ぜない
- ページタイトルでは2人の共演アイコンを使い、案内役がページ間で消えないようにする
