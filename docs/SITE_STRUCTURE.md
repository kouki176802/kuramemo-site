# サイト構造・内部リンク設計

作成日: 2026-06-25

## 目的

「選び方メモ」は、トレンド入口記事とSNS投稿から比較ページへ流す比較メディアとして運営する。

この設計では、比較ページを中心に内部リンクを組み、商品リンクは承認済み案件だけに限定する。

## 基本構造

```text
トップページ
├── AI・ガジェット
│   ├── AIツール比較
│   ├── 急速充電器比較
│   └── モバイルバッテリー比較
├── 美容
│   ├── メンズスキンケア初心者セット比較
│   └── 美容・グルーミング用品比較
├── フィットネス
│   ├── 家トレ用品比較
│   └── フィットネス補助食品比較
├── 健康
│   └── 健康補助サプリメント比較
└── 季節・暮らし
    └── 暑さ対策グッズ比較
```

## 内部リンクルール

### SNS投稿から

SNS投稿は、原則として比較ページへ送る。

例:

- AI新機能の話題 → AIツール比較
- 新型スマホの話題 → 急速充電器比較 / モバイルバッテリー比較
- 暑くなるニュース → 暑さ対策グッズ比較

### トレンド記事から

トレンド記事は、本文中盤と末尾で関連比較ページへ送る。

例:

- 「関連: 急速充電器の選び方はこちら」
- 「買う前に、比較表で対応機種と出力を確認してください」

### 比較ページから

比較ページは、関連比較ページと個別チェック記事へ送る。

例:

- 急速充電器比較 → モバイルバッテリー比較
- メンズスキンケア比較 → 暑さ対策グッズの日焼け・外出系記事
- AIツール比較 → 個別AIサービス申込前チェック
