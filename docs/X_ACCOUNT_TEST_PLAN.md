# Xアカウント初期テスト計画

作成日: 2026-06-25

対象アカウント: https://x.com/m0506k

## 目的

BOT投稿のテストを兼ねて、Xでどの投稿テーマ・投稿型が反応するか確認する。

## プロフィール案

話題の商品・サービスを比較表と選び方で整理。AI/ガジェット、美容/フィットネス、暮らし用品の「買う前チェック」を発信。広告を含む場合があります。

状態: CEOが設定済み。

## 固定投稿案

このアカウントでは、SNSやニュースで話題の商品・サービスを「買う前に見るポイント」として整理します。

扱うテーマ:

- AI・ガジェット
- 美容・フィットネス
- 季節・暮らし用品

方針:

- 全員におすすめとは書かない
- 買わなくていい人も書く
- 広告を含む場合は明記
- 未使用品を体験談のように書かない
- 比較表とチェックリスト重視

状態: CEOが設定済み。

## 初日投稿候補

### 1. AIツール

AIツールは「有名だから」で選ぶより、何の作業を減らしたいかで選ぶ方が失敗しにくいです。

文章作成、SNS投稿、画像作成、資料作成。
用途が違うと見るべき料金・機能・注意点も変わります。

※広告を含む場合があります

状態: 初回投稿READY。`samples/social/x_first_post_ready.md` に単体ファイル化済み。

投稿済みURL: https://x.com/m0506k/status/2070089999652135114

### 2. 急速充電器

急速充電器はワット数だけで選ぶと危ないです。

見るべきは、
・端末が対応しているか
・ケーブルが対応しているか
・同時充電時の出力が落ちないか

ここを外すと「高いのに速くない」が起きます。

### 3. 暑さ対策

暑さ対策グッズは「どこで使うか」で選ぶと失敗しにくいです。

室内、外出、就寝時。
用途が違うと見るべきポイントも変わります。

重さ、音、洗えるか、電源が必要かは買う前に確認。

## BOTテスト手順

```bash
python3 -m trend_commerce demo
python3 -m trend_commerce social-queue
python3 -m trend_commerce social-approve --all
python3 -m trend_commerce social-export --file output/social/x_manual_posts.csv --platform x
```

出力されたCSVを見て、Xへ手動投稿する。

実投稿APIはまだ使わない。
