# WordPress投稿計画

作成日: 2026-06-25

## 目的

WordPress導入後に、どのページをどの順番で投稿するかを決める。

## 投稿順

### 1. 固定ページ

最初に公開する。

1. トップページ
   - `site_content/homepage.md`
2. このサイトについて
   - `site_content/about.md`
3. 広告掲載方針
   - `site_content/advertising_policy.md`
4. 編集方針
   - `site_content/editorial_policy.md`
5. プライバシーポリシー
   - `site_content/privacy_policy_template.md`
6. 免責事項
   - `site_content/disclaimer.md`

### 2. カテゴリページ

1. AI・ガジェット
   - `site_content/category_ai_gadgets.md`
2. 美容・フィットネス
   - `site_content/category_beauty_fitness.md`
3. 季節・暮らし
   - `site_content/category_lifestyle_seasonal.md`

### 3. 初期比較ページ

1. AIツール比較
   - `samples/comparison_pages/001_ai_tools_comparison.md`
2. 急速充電器比較
   - `samples/comparison_pages/002_fast_chargers_comparison.md`
3. モバイルバッテリー比較
   - `samples/comparison_pages/003_mobile_batteries_comparison.md`
4. メンズスキンケア初心者セット比較
   - `samples/comparison_pages/004_mens_skincare_sets_comparison.md`
5. 家トレ用品比較
   - `samples/comparison_pages/005_home_training_items_comparison.md`
6. 暑さ対策グッズ比較
   - `samples/comparison_pages/006_heat_relief_items_comparison.md`

### 4. トレンド入口記事

比較ページ公開後に追加する。

候補:

- AIツールは何から使うべき？
- 急速充電器はどれを選ぶ？
- メンズスキンケアは何から始める？
- 家トレ用品は何から買う？
- 暑さ対策グッズは何を選ぶ？

## 下書き投稿時の注意

- 広告表記を冒頭に入れる
- 実アフィリエイトリンクがない場合は仮リンクを入れない
- 価格や在庫を断定しない
- 未使用商品を体験談にしない
- 健康効果や収益効果を断定しない
- WordPress連携は必ず `draft` で送る

## 初期公開の最低ライン

ASP審査前に最低限ほしい状態:

- 固定ページ5本
- カテゴリページ3本
- 比較ページ5〜6本
- SNS投稿10件以上
- 広告・AI利用方針
- 問い合わせ導線
