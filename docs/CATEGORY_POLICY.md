# 初期稼働カテゴリ方針

決定日: 2026-06-25

## 基本方針

この事業は一つの専門ブログに固定せず、ニュース・流行から購買機会を発見するトレンドコマース事業として運営する。ただし、立ち上げ時から全分野を公開対象にすると品質管理と案件審査が破綻するため、監視範囲と収益化範囲を分ける。

## 初期収益化カテゴリ

### 1. AI・ガジェット

- 対象: AIサービス、SaaS、スマートフォン周辺機器、充電・通信用品
- 主な購買トリガー: 新機能、新製品、規格変更、値下げ、比較需要
- 主な商品型: SaaS申込、周辺機器、サブスクリプション
- 注意: 未発表情報、リーク、誇張した生産性効果を扱わない

### 2. 美容

- 対象: メンズ美容、スキンケア、ヘアスタイリング、美容家電
- 主な購買トリガー: 新商品、季節変化、習慣開始、比較需要
- 主な商品型: 化粧品、整髪料、脱毛器、シェーバー
- 注意: 診断、治療、疾病改善、身体変化の保証を行わない

### 3. フィットネス

- 対象: 家トレ用品、プロテイン、クレアチン、アミノ酸補助食品
- 主な商品型: トレーニング器具、栄養補助食品
- 注意: 身体変化や効果を保証せず、原材料と摂取目安を明記する

### 4. 健康

- 対象: ビタミン、ミネラル、食物繊維、乳酸菌などの補助食品
- 主な商品型: 栄養機能食品、サプリメント、粉末食品
- 注意: 病気の予防・治療をうたわず、重複摂取と服薬時の注意を優先する

### 5. 季節・暮らし

- 対象: 暑さ・寒さ・梅雨・新生活・家事効率に関する一般商品
- 主な購買トリガー: 季節入り、気温・湿度変化、生活イベント
- 主な商品型: 家電、寝具、調理・収納・生活用品
- 注意: 災害、警報、被害、停電発生を販売機会として利用しない

## 監視のみ行う領域

- 転職・キャリア
- 自炊・食品
- 旅行
- エンタメ
- 海外サービス

これらはトレンドデータを保存してもよいが、案件在庫・法令・品質基準を追加するまで記事やSNS投稿を自動生成しない。

## 自動収益化禁止領域

- 医療、診断、治療、処方
- 投資助言、金融商品の推奨、暗号資産
- 災害、事故、死亡、犯罪
- 芸能人の炎上・スキャンダル
- 未成年者の不安を利用する商品
- 根拠のない健康効果、収益保証、誇大なビフォーアフター

## 拡張条件

新カテゴリは、以下をすべて満たした場合だけ収益化対象へ追加する。

1. 提携可能な案件が3件以上ある
2. SNS掲載可否と成果対象地域を確認済み
3. 公式情報源が3件以上ある
4. 禁止表現と確認項目を定義済み
5. CEOが追加を承認した

## 90日後の判定

カテゴリごとに記事数ではなく、クリック率、案件到達率、確定成果、否認率、運営工数を比較する。成果がないカテゴリは停止し、利益が確認できたカテゴリへ投稿枠を寄せる。
