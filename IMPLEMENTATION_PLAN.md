# 選び方メモ LP/比較サイト作り直し計画

作成日: 2026-06-26

## 参照

専用画像モックは未提供。今回は既存の `output/site` と、事業方針「SNS → 比較ページ → アフィリエイト」を参照として、比較メディア型LPに作り直す。

## 方向性

- 売り込み感を抑えた「買う前チェック」メディア
- SNS流入を受け止めるトップページ
- 比較ページでは広告リンクを自然に置く
- 商品名は楽天の商品名そのままではなく、見出しとして読みやすく整形
- 広告表記、免責、編集方針を目立つ場所に置く

## 実装対象

- 既存の静的サイト生成器 `trend_commerce/static_site.py`
- 出力先 `output/site`
- 楽天商品選定BOTが入れた `data/offers.csv`

## 受け入れ基準

- トップページにHero、信頼導線、カテゴリカード、注目比較記事、運用方針がある
- 比較ページの商品CTAがカード化され、広告リンクであることが分かる
- 長すぎる楽天商品名を表示用に短縮する
- PC/スマホでテキストが重ならない
- CTAに控えめなホバー/アニメーションがある
- `prefers-reduced-motion` に対応する
- ローカルブラウザでトップ・比較ページ・モバイル幅を確認する
- テストが通る
